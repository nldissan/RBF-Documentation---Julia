The Radial Basis Function Method (RBF) in Julia - Documentation
==================================================

Getting started
=====================

The purpose of this documentation is to share the RBF method codes written in Julia language. The codes include node generation, interpolation, and discretizing differential operators related to the RBF method.  However, the documentation does not provide instructions on how to install or set up Julia. The following link provides basic information about installing Julia. 

* `Install Julia <https://julialang.org/downloads/>`_
